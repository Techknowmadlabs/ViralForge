# Phase 5: Scriptwriting System

## Purpose

Produce production-ready scripts that are engineered for retention, not just information delivery. Every script is structured for the ear (spoken word), tagged with visual cues for the production pipeline, and optimized for platform-specific audience behavior.

---

## Retention Architecture

The script is a retention machine. Every structural element exists to keep the viewer watching.

### The Retention Stack

```
LAYER 1: MACRO STRUCTURE (overall arc)
├── Hook (0-30 seconds) — Open loop that demands resolution
├── Setup (30 sec - 2 min) — Context + promise of value
├── Body (2 min - [length-2] min) — Deliver value in escalating chunks
├── Climax ([length-2] - [length-1] min) — The payoff / main revelation
└── Close (final 60 seconds) — CTA + final thought + tease next video

LAYER 2: MESO STRUCTURE (within the body)
├── Segment 1 (90-120 seconds) — First major point
│   ├── Pattern Interrupt
│   └── Micro-payoff
├── Segment 2 (90-120 seconds) — Second major point
│   ├── Pattern Interrupt
│   └── Micro-payoff
├── Segment 3 (90-120 seconds) — Third major point
│   ├── Pattern Interrupt
│   └── Micro-payoff
└── [Continue as needed for video length]

LAYER 3: MICRO STRUCTURE (sentence level)
├── Vary sentence length (short punchy, then longer explanatory)
├── Use "you" and "your" every 3-4 sentences (direct address)
├── Rhetorical questions every 60 seconds
└── Transition phrases that create forward momentum
```

## Hook Engineering

The hook has 8 seconds to stop the scroll. These are the proven hook architectures:

### Hook Type 1: The Shocking Stat
```
"[Dramatic statistic] — and [implication that affects the viewer]."
```
Example: "93% of YouTube channels never reach 1,000 subscribers — and the reason has nothing to do with content quality."

### Hook Type 2: The Contrarian Statement
```
"Everything you've been told about [topic] is wrong. Here's what [authority/data] actually shows."
```
Example: "Everything you've been told about diversifying your portfolio is wrong. Here's what the data from the last 10 years actually shows."

### Hook Type 3: The Story Open
```
"[Specific time marker], [vivid scene], [and then something unexpected happened]."
```
Example: "Last Tuesday, at 3 AM, a single line of code crashed a $4 billion hedge fund — and nobody noticed for six hours."

### Hook Type 4: The Direct Challenge
```
"If you [common behavior], you're [negative consequence] — and I can prove it in [time]."
```
Example: "If you're still using a traditional savings account, you're losing $4,000 a year — and I can prove it in the next 8 minutes."

### Hook Type 5: The Result Reveal
```
"I [did specific thing] and [unexpected result]. [Brief tease of how]."
```
Example: "I replaced my entire marketing team with AI tools and revenue went up 340%. Here's exactly how I did it."

### Hook Type 6: The Curiosity Gap
```
"There's a [thing] that [impressive claim], but [surprising detail that demands explanation]."
```
Example: "There's a free tool that can predict stock prices with 78% accuracy, but Wall Street doesn't want you to know about it."

### Hook Selection Guide

| Video Type | Best Hook Types | Why |
|-----------|----------------|-----|
| Tutorial | Type 4 (Challenge) or Type 1 (Stat) | Creates urgency to learn |
| Analysis | Type 2 (Contrarian) or Type 1 (Stat) | Challenges assumptions |
| Story | Type 3 (Story Open) or Type 5 (Result) | Emotional engagement |
| List/Ranking | Type 1 (Stat) or Type 6 (Curiosity) | Sets stakes for the list |
| Comparison | Type 2 (Contrarian) or Type 4 (Challenge) | Creates a "side" to root for |
| News/Reaction | Type 3 (Story) or Type 1 (Stat) | Immediacy and stakes |

## Pattern Interrupts

Viewer attention decays. Pattern interrupts reset it. Place one every 60-90 seconds.

### Pattern Interrupt Types

| Type | How It Works | Script Tag |
|------|-------------|-----------|
| **Visual Shift** | Change the visual style suddenly | `[VISUAL: SHIFT — change to X style]` |
| **Direct Address** | Break fourth wall, speak directly to viewer | "Now, I know what you're thinking..." |
| **Sound Design** | Unexpected sound effect or music change | `[AUDIO: SFX — dramatic sting]` |
| **Zoom/Scale** | Dramatic zoom in or out | `[VISUAL: ZOOM — tight close-up]` |
| **Counter/Graphic** | On-screen number, countdown, or graphic | `[VISUAL: GRAPHIC — animated counter]` |
| **Humor** | Brief joke, meme reference, or absurd comparison | Write the joke into the narration |
| **Mini-Story** | 15-second anecdote or example | Start with "Here's the thing..." or "Let me give you an example." |
| **Prediction Prompt** | Ask viewer to guess before revealing | "Before I tell you the answer, take a guess." |

## Writing for the Ear

Scripts written for video narration follow different rules than written content.

### Rules

1. **Read every sentence aloud before committing it** — if it's awkward to say, rewrite it
2. **Contractions are mandatory** — "you're" not "you are", "it's" not "it is", "can't" not "cannot"
3. **Short sentences dominate** — Average sentence length: 8-12 words. Occasional long sentence for rhythm: 15-20 words maximum
4. **Active voice always** — "The algorithm rewards consistency" not "Consistency is rewarded by the algorithm"
5. **Concrete over abstract** — "$4,000 per year" not "significant financial loss"
6. **Avoid jargon without context** — When introducing a technical term, immediately define it: "The CPM — that's how much advertisers pay per thousand views — in this niche is..."
7. **Use "you" every 3-4 sentences** — Direct address maintains connection
8. **One idea per sentence** — Never compound clauses in narration
9. **Vary rhythm** — Short. Short. Then a longer sentence that provides the context and the evidence. Short.

### Word Count Targets

| Video Length | Target Word Count | Words Per Minute |
|-------------|-------------------|-----------------|
| 3 minutes | 450-500 words | ~155 |
| 5 minutes | 725-800 words | ~155 |
| 8 minutes | 1,150-1,300 words | ~155 |
| 10 minutes | 1,450-1,600 words | ~155 |
| 15 minutes | 2,200-2,400 words | ~155 |
| 20 minutes | 2,900-3,200 words | ~155 |

## Platform-Specific Script Adaptations

### YouTube Shorts Script (30-60 seconds)

```markdown
[VISUAL: Opening frame — must hook in 1 second]
[TEXT ON SCREEN: Hook text, 5-7 words max]

NARRATION (15-20 seconds):
"[Deliver one single insight with maximum impact]"

[VISUAL: Proof/evidence visual]
[TEXT ON SCREEN: Key stat or result]

NARRATION (10-15 seconds):
"[Explain the 'so what' — why this matters to the viewer]"

[VISUAL: Loop point — visual that connects back to the opening]
[TEXT ON SCREEN: CTA text]

NARRATION (5-10 seconds):
"[End with a loop or cliffhanger that encourages rewatch]"
```

### Instagram Reel / TikTok Script (15-60 seconds)

```markdown
[VISUAL: Scroll-stopping opening frame]
[TEXT: Bold on-screen text with hook]
[AUDIO: Trending sound or original audio]

NARRATION/TEXT (entire video is text-driven or voice-driven):
"[Deliver maximum value in minimum time. One core message. Visual storytelling. End with engagement trigger (save this, share with someone who needs it, comment your take).]"
```

### X/Twitter Video Script (30-120 seconds)

```markdown
TWEET TEXT: "[Provocative statement or question that frames the video. This is as important as the video itself — the tweet drives the click.]"

VIDEO:
[VISUAL: Opinion-driven, personality-forward, raw/authentic feel]

NARRATION:
"[Take a strong position. X rewards opinion and debate. Every video should make someone want to reply — whether they agree or disagree.]"

THREAD CONTINUATION:
"[Plan 2-3 follow-up tweets that expand on the video's point]"
```

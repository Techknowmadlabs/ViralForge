# Phase 7: Platform Operations & Cross-Platform Strategy

## Purpose

Optimize content for each platform's unique algorithm, audience behavior, and technical requirements. One piece of content becomes 5-7 platform-native assets — not lazy cross-posts, but genuinely adapted content.

---

## Platform Technical Specifications

### YouTube Long-Form

| Spec | Requirement |
|------|-------------|
| Resolution | 1920x1080 (16:9) minimum, 3840x2160 (4K) preferred |
| Frame Rate | 24fps (cinematic), 30fps (standard), 60fps (tutorial/gaming) |
| Format | MP4 (H.264), MOV |
| Max File Size | 256 GB or 12 hours |
| Thumbnail | 1280x720 (16:9), JPG/PNG, under 2MB |
| Title Length | Under 60 characters (70 max, but truncates on mobile) |
| Description | 5,000 character limit |
| Tags | 500 character total limit |
| Chapters | Minimum 3 chapters, first at 0:00 |
| End Screens | Last 20 seconds, max 4 elements |
| Cards | Max 5 per video |
| Optimal Length | 8-15 minutes (monetization sweet spot) |

### YouTube Shorts

| Spec | Requirement |
|------|-------------|
| Resolution | 1080x1920 (9:16) |
| Max Length | 3 minutes (60 seconds optimal for new channels) |
| Format | MP4, vertical only |
| Thumbnail | Auto-selected from video (can set manually) |
| Captions | Burned in, not SRT (platform adds its own) |
| Sound | Critical — Shorts shelf is sound-on by default |
| Loop Point | Design the end to seamlessly loop to the beginning |

### Instagram Reels

| Spec | Requirement |
|------|-------------|
| Resolution | 1080x1920 (9:16) |
| Max Length | 90 seconds (15-30 seconds optimal for algorithm) |
| Format | MP4 |
| Caption Length | 2,200 characters max |
| Hashtags | 5-10 (less is more, relevance over volume) |
| Cover Image | 1080x1920, can be custom |
| Audio | Trending audio significantly boosts reach |
| Captions | Burned-in text captions mandatory |

### TikTok

| Spec | Requirement |
|------|-------------|
| Resolution | 1080x1920 (9:16) |
| Max Length | 10 minutes (15-60 seconds optimal) |
| Format | MP4, MOV |
| Caption Length | 4,000 characters |
| Hashtags | 3-5 (algorithm weighs them heavily) |
| Sound | Essential — use trending sounds when possible |
| Text | On-screen text required for accessibility and engagement |
| Stitch/Duet | Enable for engagement, use for content creation |

### X (Twitter) Video

| Spec | Requirement |
|------|-------------|
| Resolution | 1280x720 minimum, 1920x1080 preferred |
| Aspect Ratios | 16:9 (landscape), 1:1 (square), 9:16 (portrait) all supported |
| Max Length | 2:20 (free), 25 minutes (Premium), up to 3 hours (Premium+) |
| Format | MP4 |
| File Size | 512 MB max |
| Tweet Length | 280 characters (attach video separately) |
| Thread | Use threads to add context and drive engagement |

### Facebook Video

| Spec | Requirement |
|------|-------------|
| Resolution | 1280x720 minimum |
| Aspect Ratios | 16:9 (feed), 9:16 (Reels), 1:1 (square, highest engagement) |
| Max Length | 240 minutes |
| Format | MP4 |
| Caption Text | 63,206 characters max (but keep under 200 for engagement) |
| Captions | Auto-generated or SRT upload |
| Optimal Length | 3-5 minutes for feed, 15-60 seconds for Reels |

### LinkedIn Video

| Spec | Requirement |
|------|-------------|
| Resolution | 1920x1080 preferred |
| Aspect Ratios | 16:9 (landscape), 1:1 (square), 9:16 (portrait) |
| Max Length | 15 minutes (native), 10 minutes (ads) |
| Format | MP4 |
| Caption Length | 3,000 characters |
| Optimal Length | 1-3 minutes |
| Tone | Professional, insight-driven, career-relevant |

## Content Adaptation Matrix

The same video topic manifests differently per platform. Never just crop and re-upload.

### Adaptation Framework

```
MASTER CONTENT (YouTube long-form, 10 minutes)
│
├── YouTube Short (30-60 sec)
│   └── Extract the single most surprising moment
│   └── Add hook text on screen in first frame
│   └── Loop-friendly ending
│
├── Instagram Reel (15-30 sec)
│   └── Visual-first storytelling (less narration, more on-screen text)
│   └── Trending audio overlay
│   └── Aesthetic emphasis (brand colors prominent)
│   └── CTA: "Save this for later" or "Share with someone who needs this"
│
├── TikTok (15-60 sec)
│   └── Casual, native feel (not overly produced)
│   └── Trending sound or format
│   └── On-screen text carrying the narrative
│   └── Comment-bait ending ("What would you do?")
│
├── X/Twitter (30-120 sec)
│   └── Opinion-led: take a strong position
│   └── Thread format: video + 3-5 tweet thread expanding on the topic
│   └── Engagement hook: ask a question that invites replies
│
├── Facebook (3-5 min)
│   └── Shareable format: "tag someone who needs to see this"
│   └── Emotional hook (Facebook audience responds to emotion > information)
│   └── Square format for feed (higher engagement than 16:9)
│   └── Captions essential (Facebook autoplay is muted)
│
└── LinkedIn (1-3 min)
    └── Professional framing of the same topic
    └── Career or business relevance angle
    └── Thought leadership positioning
    └── Text post with key takeaways + video embedded
```

### Per-Platform Hook Adaptation

The same topic needs platform-native hooks:

| Platform | Hook Style | Example (topic: "AI replacing jobs") |
|----------|-----------|--------------------------------------|
| YouTube | Curiosity + value promise | "These 7 jobs won't exist in 3 years — and the replacements are already here." |
| Shorts | Shock + speed | "Your job is being replaced. Here's proof." |
| Instagram | Aesthetic + save-worthy | "🔮 The future of work in 30 seconds (save this)" |
| TikTok | Trend-native + debate | "POV: You just got replaced by AI at your job" |
| X/Twitter | Opinion + provocation | "Unpopular opinion: If AI can do your job, you were already replaceable." |
| Facebook | Emotional + shareable | "My friend just lost his 15-year career to an AI tool. Here's what happened." |
| LinkedIn | Professional + actionable | "I've been studying AI's impact on 50 industries. Here are the 3 things every professional should do now." |

## Community Management Protocol

Growth requires engagement, not just posting.

### First 30 Minutes After Upload

1. **Reply to every comment** for the first hour — this signals the algorithm that the content is generating discussion
2. **Pin a comment** with additional value (resource link, related video, or question to drive more comments)
3. **Heart comments** that are substantive
4. **Share to secondary platforms** with platform-native captions

### Ongoing Community Rules

- Reply to comments within 24 hours (at minimum)
- Ask a question in the pinned comment to drive engagement
- Create polls (Community tab, Instagram Stories, X) weekly
- Respond to negative comments professionally — they drive algorithm engagement too
- Cross-reference other creators (builds network, drives collaboration potential)

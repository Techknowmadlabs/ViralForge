## What is ViralForge?

ViralForge is an **enterprise-grade Claude Code skill** that transforms a single content idea into a fully optimized, multi-platform video content operation. It orchestrates an 8-phase pipeline covering every aspect of video content creation — from market intelligence and brand identity through scriptwriting, SEO optimization, platform-native adaptation, and data-driven iteration.

**This is not a template collection.** It's an autonomous content engine with:

- **7 production-ready Python CLI tools** (zero external dependencies)
- - **8 comprehensive reference playbooks** (50,000+ words of battle-tested strategy)
  - - **Cross-platform adaptation** for 6 major social platforms
    - - **Built-in A/B testing frameworks** and analytics feedback loops

## What is ViralForge?

ViralForge is an **enterprise-grade Claude Code skill** that transforms a single content idea into a fully optimized, multi-platform video content operation. It orchestrates an 8-phase pipeline covering every aspect of video content creation.

**This is not a template collection.** It is an autonomous content engine with:

- **7 production-ready Python CLI tools** (zero external dependencies)
- **8 comprehensive reference playbooks** (50,000+ words of battle-tested strategy)
- **Cross-platform adaptation** for 6 major social platforms
- **Built-in A/B testing frameworks** and analytics feedback loops

---

## The 8-Phase Pipeline

| Phase | Name | What It Does | Script |
|-------|------|-------------|--------|
| 1 | **Niche Intelligence** | Scores niches across 6 dimensions | niche_analyzer.py |
| 2 | **Visual Brand System** | Generates complete visual identity | — |
| 3 | **Content Strategy** | Builds 90-day content calendars | content_calendar.py |
| 4 | **SEO and Discovery** | Title variants, descriptions, tags | seo_optimizer.py |
| 5 | **Scriptwriting Engine** | Retention-optimized scripts | script_generator.py |
| 6 | **Production Pipeline** | Tool chains, SOPs, cost tiers | — |
| 7 | **Platform Operations** | Cross-platform adaptation | platform_adapter.py |
| 8 | **Analytics and Iteration** | KPIs, A/B testing, monetization | — |

**Orchestrator:** pipeline.py runs all phases in sequence.

## Quick Start

### Install as Claude Code Skill

Copy the .skill file to your Claude Code skills directory, then ask Claude to run the pipeline.

### Use Python CLI Tools Directly

All scripts use Python 3.10+ stdlib only — zero pip install required.

**Analyze a niche:**
```bash
python scripts/niche_analyzer.py --niches "AI tools,personal finance,health tech"
```

**Generate a 90-day calendar:**
```bash
python scripts/content_calendar.py --niche "AI tools" --platforms youtube,instagram,tiktok --budget 50
```

**Run the full pipeline:**
```bash
python scripts/pipeline.py --niche "AI tools" --platforms youtube,instagram,tiktok,x --budget 50 --titles "7 AI Tools|5 Prompts That 10x Productivity"
```
